Онлайн Кнопка

Первый нажавший видит черный квадрат, остальные не могут нажать

Загрузка...

Как это работает: Кнопка доступна всем, но только первый пользователь, который нажмет её, увидит черный квадрат. После этого кнопка станет неактивной для всех остальных пользователей.

Работает в реальном времени для всех пользователей
